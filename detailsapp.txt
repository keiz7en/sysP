================================================================================
                           1. SYSTEM OVERVIEW
================================================================================

EduAI is a comprehensive, production-ready Education Management System with 
integrated Google Gemini AI features for personalized learning, career guidance, 
automated assessments, and real-time student-teacher-admin collaboration.

KEY ARCHITECTURE:
- Frontend: React 18 with TypeScript, Vite bundler, Framer Motion animations
- Backend: Django 5.0 REST Framework with 10 specialized apps
- AI Engine: Google Gemini AI (gemini-1.5-flash-latest model)
- Database: SQLite (production-ready, 956KB with real data)
- Authentication: Django Token Authentication with role-based access control
- Real-time: WebSocket support for chat and notifications

┌─────────────────────────────────────────────────────────────────────┐
│                          TEACHER WORKFLOW                           │
└─────────────────────────────────────────────────────────────────────┘

STEP 1: REGISTRATION & APPROVAL
1. Teacher registers with credentials
2. Admin receives approval request
3. Admin approves → Teacher status = "active"

STEP 2: REQUEST TEACHING SUBJECTS
1. Teacher requests subjects they want to teach
2. Admin reviews qualifications
3. Admin approves → TeacherApprovedSubject created

STEP 3: CREATE COURSES
1. Teacher creates course under approved subject
2. Course includes: title, code, description, credits, syllabus
3. Course status automatically "active"
4. AI features enabled by default

STEP 4: CREATE ASSIGNMENTS
1. Teacher creates assignments for each course
2. Assignment types: Quiz, Homework, Project, Exam
3. Set due dates, max points, instructions
4. Students see assignments after enrollment approval

STEP 5: MANAGE STUDENT ENROLLMENTS
1. View enrollment requests from students
2. Approve/reject based on prerequisites
3. Approved students get:
   - Access to course materials
   - Can submit assignments
   - AI features unlocked
   - Progress tracking activated

STEP 6: GRADE STUDENT WORK (NEW FEATURE)
1. View all pending submissions in review queue
2. See AI-suggested grades (auto-marked)
3. Adjust marks and add feedback
4. Publish grades → Students notified
5. Automatic progress % calculation

STEP 7: PUBLISH FINAL GRADES
1. After course completion (progress = 100%)
2. System calculates final grade:
   - Weighted average of all assignments
   - Letter grade (A, B, C, D, F)
   - GPA calculation (4.0 scale)
3. Grade appears in student transcript
4. Student can now rate teacher

STEP 8: VIEW ANALYTICS
1. Student performance trends
2. Course completion rates
3. Average grades per assignment
4. Dropout predictions
5. Engagement metrics

ACCESS:
- URL: http://localhost:3000/teacher
- Can only see their own courses
- Cannot modify other teachers' content

Student can now access 11 AI-powered features:

   - Upload transcript or enter GPA
   - AI analyzes academic performance
   - Get strength/weakness insights
   - Receive improvement recommendations

B) PERSONALIZED LEARNING CONTENT
   - Select course and topic
   - AI generates custom study materials
   - Adaptive difficulty based on progress
   - Practice problems with solutions

C) AI QUIZ GENERATOR
   - Choose subject and difficulty
   - AI generates quiz questions
   - Immediate feedback
   - Explanations for wrong answers

D) CAREER GUIDANCE
   - Input skills and interests
   - AI suggests career paths
   - Job market insights
   - Skill gap analysis
   - Training recommendations

E) RESUME ANALYSIS
   - Upload resume (PDF/TXT)
   - AI evaluates format, content, keywords
   - Get ATS optimization tips
   - Improvement suggestions

F) ESSAY GRADING
   - Paste essay or assignment
   - Provide rubric/criteria
   - AI grades and provides feedback
   - Grammar, structure, content analysis

G) PERFORMANCE PREDICTION
   - AI predicts final grade based on current performance
   - Risk of dropout detection
   - Early intervention alerts

H) 24/7 CHATBOT ADVISOR
   - Ask academic questions anytime
   - Get study tips
   - Clarify course concepts
   - Exam preparation help

I) ENGAGEMENT ANALYSIS
   - Track study time and activity
   - Get engagement score
   - Motivation recommendations

J) FEEDBACK ANALYSIS
   - AI analyzes teacher feedback patterns
   - Identifies recurring issues
   - Action plan for improvement

K) COMPREHENSIVE AI DASHBOARD
   - All AI insights in one place
   - Visual analytics
   - Progress charts

STEP 6: COMPLETE ASSIGNMENTS
1. View assignment details
2. Use AI assistant for help (optional)
3. Submit work through system
4. Submission status: "submitted"
5. Wait for teacher grading

STEP 7: RECEIVE GRADES
1. Teacher grades submission
2. Student receives notification
3. View marks, feedback, rubric
4. Progress % automatically updates
5. Formula: (completed_tasks / total_tasks) * 100

STEP 8: COURSE COMPLETION
1. When progress reaches 100%
2. System calculates final grade
3. Grade published to transcript
4. Course status: "completed"
5. Student can now rate teacher

STEP 9: RATE TEACHER (OPTIONAL)
1. After course completion
2. Give 1-5 star rating
3. Write feedback/review
4. Submission is anonymous
5. Teacher's average rating updates

STEP 10: VIEW TRANSCRIPT
1. See all completed courses
2. View all grades and GPAs
3. Calculate overall GPA
4. Download transcript PDF

FILE: backend/courses/progress_service.py (357 lines) ✨ NEW

┌────────────────────────────────────────────────────────────────────┐
│ AUTOMATIC PROGRESS CALCULATION                                     │
└────────────────────────────────────────────────────────────────────┘

FORMULA:
progress_percentage = (completed_assignments / total_assignments) × 100

TRIGGERS:
1. When student submits assignment → progress updates
2. When teacher grades submission → progress updates
3. When new assignment created → progress recalculates for all students

PROCESS:
1. Count total assignments for course
2. Count student's graded assignments
3. Calculate percentage
4. Update CourseEnrollment.progress field
5. Check if progress = 100% → trigger course completion

┌────────────────────────────────────────────────────────────────────┐
│ GRADING WORKFLOW                                                   │
└────────────────────────────────────────────────────────────────────┘

FILE: backend/courses/progress_service.py → GradingService class

STEP 1: Student Submits Assignment
- AssignmentSubmission created
- Status: "submitted"
- Submission time recorded
- Teacher notified

STEP 2: Teacher Views Submission
- GET /api/teachers/grading/review-queue/
- Returns all pending submissions
- Sorted by due date

STEP 3: Teacher Grades Submission
- POST /api/teachers/grading/grade-submission/
- Payload:
  {
    "submission_id": 123,
    "marks_obtained": 85,
    "feedback": "Great work!",
    "rubric_scores": {
      "content": 30,
      "structure": 25,
      "grammar": 30
    }
  }
- Submission status: "submitted" → "graded"
- Progress automatically recalculated

STEP 4: Progress Updates
- Backend calls: ProgressTrackingService.update_progress_on_submission_grade()
- Recalculates completion percentage
- Updates CourseEnrollment.progress
- If progress = 100% → trigger completion

STEP 5: Course Completion Triggered
- Backend calls: ProgressTrackingService.trigger_course_completion()
- Checks all assignments are graded
- Calculates final grade (see below)
- Creates Grade record
- Marks enrollment as completed

┌────────────────────────────────────────────────────────────────────┐
│ FINAL GRADE CALCULATION                                            │
└────────────────────────────────────────────────────────────────────┘

FORMULA:
weighted_average = Σ(marks_obtained × assignment_weight) / Σ(max_points × assignment_weight)
final_score = weighted_average × 100

LETTER GRADE CONVERSION:
- A: 90-100%
- B: 80-89%
- C: 70-79%
- D: 60-69%
- F: 0-59%

GPA CONVERSION (4.0 scale):
- A: 4.0
- B: 3.0
- C: 2.0
- D: 1.0
- F: 0.0

PROCESS:
1. Fetch all graded submissions for student in course
2. Calculate weighted average based on assignment max_points
3. Convert to percentage
4. Assign letter grade
5. Calculate GPA points
6. Create Grade record with breakdown
7. Update student's overall GPA
8. Publish grade → student notified

```python
def update_average_rating(self):
    ratings = TeacherRating.objects.filter(teacher=self)
    if ratings.exists():
        avg = ratings.aggregate(Avg('rating'))['rating__avg']
        self.average_rating = round(avg, 2)
        self.total_ratings = ratings.count()
        self.save()
```
